# Ecommerce-analytics-dashboard
An interactive Power BI dashboard analyzing e-commerce sales performance, revenue trends, customer behavior , and key business KPI's.

# Overview 
This dashboard analyzes sales performance of an e-commerce business.

Average Order Value (AOV)
Cancellation Rate
Total Orders
